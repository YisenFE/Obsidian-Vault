# OpenAI

## 在本 wiki 中的角色

OpenAI 是 [[wiki/entities/products/Codex|Codex]] 背后的组织，并发布了 Codex-first engineering 来源；在 LangChain 来源中，OpenAI 也作为 [[wiki/entities/products/GPT-5.2-Codex|GPT-5.2-Codex]] 的模型组织出现。

## 来源支持的要点

- OpenAI 来源描述了一个内部 Codex-first 软件工程实验。
- OpenAI 的 framing 强调 harness、反馈回路、仓库本地上下文和机械约束，认为这些是 agent-first engineering 的难点。
- LangChain 来源使用 GPT-5.2-Codex 作为固定模型，再优化周围 harness。

- Michael Bolin 访谈补充了 OpenAI Codex 的 builder-side harness philosophy：模型体验主导上限，但 harness 需要足够小、可靠、安全，并通过 sandbox 和工具编排支撑真实机器上的 agent execution。
