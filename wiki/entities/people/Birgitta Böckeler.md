# Birgitta Böckeler

## 在本 wiki 中的角色

[[wiki/sources/2026-05-28 Harness engineering for coding agent users|Harness engineering for coding agent users]] 的作者。该文在 MartinFowler.com 上提出 coding agent 用户侧 harness engineering 的框架。

## 来源支持的要点

- 她把 coding-agent user harness 定义为用户围绕特定系统搭建的外层生成前引导和反馈控制。
- 她提出 feedforward guides、feedback sensors、computational/inferential controls、regulation categories、harnessability 与 harness templates 等概念。
- 文章说明研究和写作中使用了 GenAI，包括 Claude 与 Claude Code，用于从笔记中提取想法和润色语言。
