# Zed

## 在本 wiki 中的角色

Zed 是来源作者偏好的 Claude Code 搭配编辑器。

## 来源支持的要点

- 来源称 Zed 提供 Claude integration / agent panel、实时文件跟踪、命令面板、低资源占用、Vim mode 和 file watcher。
- 作者建议 split screen：一侧 terminal 跑 Claude Code，一侧编辑器审阅文件。
