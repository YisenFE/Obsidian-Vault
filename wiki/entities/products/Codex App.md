# Codex App

## 在本 wiki 中的角色

Codex App 是访谈中提到的 Codex 新界面，被 Bolin 类比为 mission control，用于管理多条并行 agent conversations。

## 来源支持的要点

- Bolin 说 Codex App 让开发者浏览 agent diff、通过 command J 打开 terminal，并跨多个 agents 组织工作。
- 该界面体现了从单 IDE 工作流转向多 agent 并行 workstreams 的变化。
