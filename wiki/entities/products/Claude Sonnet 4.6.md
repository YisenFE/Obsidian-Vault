# Claude Sonnet 4.6

## 在本 wiki 中的角色

Claude Sonnet 4.6 是 Agents that remember workshop 与 Claude dreams docs 中提到的 Anthropic 模型之一，可用于运行 dreaming pipeline，通常作为质量/成本取舍中的较轻模型选择。

## 来源支持的要点

- Workshop 提到 dream job 可选择 Opus 4.7 或 Sonnet 4.6，取决于质量要求和 token 成本。
- 官方 docs 补充 dreaming research preview 支持 Claude Opus 4.8、Claude Opus 4.7 和 Claude Sonnet 4.6。
