# Deep Agents

## 在本 wiki 中的角色

Deep Agents 是 LangChain 的 open-source coding agent / CLI；在 [[wiki/sources/2026-05-28 Improving Deep Agents with harness engineering|Improving Deep Agents with harness engineering]] 中，它是被优化 harness 的 agent。

## 来源支持的要点

- 来源把 `deepagents-cli` 称为 LangChain 的 coding agent。
- 文章报告在固定 GPT-5.2-Codex 的情况下，仅通过改变 harness 就提升 Terminal Bench 2.0 表现。
- 来源称 Deep Agents 是 open source，并链接 Python 与 JavaScript 实现。
- The Anatomy 文章结尾把 harness engineering 连接到 LangChain 的 Deep Agents library，说明这些 harness primitive 是其持续研究和产品化的对象。
