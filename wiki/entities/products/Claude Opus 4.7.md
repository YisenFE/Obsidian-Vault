# Claude Opus 4.7

## 在本 wiki 中的角色

Claude Opus 4.7 是 Agents that remember workshop 与 Claude dreams docs 中提到的 Anthropic 模型之一，可用于运行 dreaming pipeline。

## 来源支持的要点

- Workshop 创建 dream job 时选择 Opus 4.7，并说明也可按质量和 token 成本在 Opus 4.7 与 Sonnet 4.6 之间选择。
- 官方 docs 补充 dreaming research preview 支持 Claude Opus 4.8、Claude Opus 4.7 和 Claude Sonnet 4.6。
