# Claude Platform

## 在本 wiki 中的角色

Claude Platform 是 Anthropic 面向开发者的产品平台；Agents that remember 来源中的 Claude Managed Agents、Memory Stores、Dreaming、Console 和 CLI 都属于该平台能力。

## 来源支持的要点

- Workshop 通过 CLI 和 Console 演示创建 session、memory store、dream job，以及查看 memory/diff/status。
- 官方 docs 将 memory stores 和 dreams 放在 Managed Agents 的 “Build persistent memory” 路径下。
