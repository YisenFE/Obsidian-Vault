# Model Context Protocol

## 在本 wiki 中的角色

Model Context Protocol（MCP）是来源中用来把外部服务连接到 Claude Code 的协议/服务器体系。

## 来源支持的要点

- 来源举例包括 Supabase、GitHub、Vercel、Railway、Cloudflare、ClickHouse、Firecrawl、memory、sequential-thinking 等 MCP servers。
- 来源强调 MCP 更像 prompt-driven wrapper，不是普通 API 的替代品。
- 来源也强调 MCP 必须做 context budget 管理，不使用的 server 应禁用。
