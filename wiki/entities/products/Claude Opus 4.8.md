# Claude Opus 4.8

## 在本 wiki 中的角色

Claude Opus 4.8 是 Claude Code dynamic workflows 来源中提到的 Anthropic 模型，用来说明模型能力提升后，Claude 已能为具体任务动态编写 custom harness。

## 来源支持的要点

- 来源把 Claude Opus 4.8 与 dynamic workflows 放在一起，说明 Claude 能写出 tailor-made harness，而不是只能依赖静态 workflow。
- 该说法是来源时点的官方产品叙述，本 wiki 未独立验证模型能力或版本行为。
- Claude dreams docs 补充 Opus 4.8 是 dreaming research preview 支持的模型之一。
