# tmux

## 在本 wiki 中的角色

tmux 是来源中用于长命令、服务进程和日志观察的终端复用工具。

## 来源支持的要点

- 来源建议在 Claude Code 执行长时间命令、启动 frontend/backend servers 时使用 tmux，便于 detach/attach 和查看日志。
- Hook 示例会在长命令前提醒如果不在 TMUX 中，应考虑使用 tmux。
