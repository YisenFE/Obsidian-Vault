# VS Code

## 在本 wiki 中的角色

VS Code 是访谈中提到的 Codex 集成界面之一；Bolin 称 VS Code extension 后用户明显转向该界面。

## 来源支持的要点

- 来源称 VS Code extension 在 2025 年夏秋后推出，并相信 VS Code usage 超过 CLI usage。

- Claude Code shorthand 来源把 VS Code / Cursor 作为 Claude Code 可搭配编辑器，提到可通过 `\ide` 同步启用 LSP 功能，也可使用更集成的 VS Code extension。
- 该来源截图来自 Claude Code docs，强调 VS Code extension 提供 inline diffs、@-mentions、plan review 和键盘快捷键等体验；本 wiki 未独立核验当前文档状态。
